# Steps for Generation

## ebook

This book requires that you have [asciidoc-epub](https://docs.asciidoctor.org/epub3-converter/latest/) installed.

``` sh
asciidoctor-epub3 sinners_in_the_hands_of_an_angry_god.adoc
```

## pdf

``` sh
typst compile sinners-in-the-hands-of-an-angry-god.typ sinners-in-the-hands-of-an-angry-god.pdf
```
